Нажмите кнопку для создания персонализированного образа.
Образ получит уникальный hostname и будет готов к записи на SD карту.

Ручная сборка

1
Артефакты (~15 мин)
bash 00-build-arm64-artifacts-trixie.sh
2
Базовый образ (1-2 часа)
sudo ARTIFACTS_DIR=./artifacts-arm64 bash 01-build-base-pi-image-trixie.sh
3
Запись на SD
Linux: sudo dd if=image.pony.img of=/dev/sdX bs=4M status=progress
Windows: Rufus / balenaEtcher / Raspberry Pi Imager
